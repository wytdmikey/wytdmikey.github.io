installation instructions if you need it!

here's how to set this repository up with your Kodi build:

  • 1. go to the Kodi file manager.
  • 2. click on "Add source".
  • 3. where it says "None", you want to hit the pencil icon next to it. it'll bring up a keyboard that'll say "Enter the paths or browse for the media locations". simply punch in https://wytdmikey.github.io/packages.
  • 4. you can name it whatever you want as it won't affect anything in any way, however i personally like to name it "mikey's fun house repository!" and keep it pushing.
  • 5. back out and head to Kodi's add-on menu.
  • 6. select "Install from zip file". a menu will pop up where all your repositories/sources are located.
  • 7. select my repository, and install repository.mikey-1.0.zip. this would actually install the repository onto your Kodi build.
  • 8. back out but remain in the add-on menu. select "Install from repository", and then from there, select my repository.
  • 9. install whatever the fuck you want from there.
enjoy!
